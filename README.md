# Pure Html

## Features

* Hot-reload html and css.
* Inline styling.
* Html validation.
* Autoprefixer.

## Content Creation

### Config

Order in which options are set:
1. Settings.
2. Local config file.
3. Global config file.

* Follows [Conventional Commits](https://conventionalcommits.org/).
